# Gestión Local 📦

Sistema de gestión para negocios, desarrollado para optimizar la carga de productos y el control de inventario.

## 🚀 Tecnologías
- **Backend:** NestJS, Prisma ORM, PostgreSQL
- **Frontend:** Next.js
- **Herramientas:** Git, Postman

## 🛠 Características
- Gestión de productos (CRUD completo).
- Categorización de inventario.
- Arquitectura monorepo.
